Weather App (SwiftUI + Firebase)

Weather App built with SwiftUI, Open-Meteo API, and Firebase Realtime Database.
The app allows users to search for cities, view current weather and forecasts, and manage favorite cities with notes.
Features
Assignment 7 – Weather

City search with suggestions (Geocoding)

Current weather: temperature, condition, humidity, feels like, wind

24-hour hourly forecast

Celsius / Fahrenheit switch

Offline mode with cached last response

Assignment 8 – Firebase

Firebase Authentication (Anonymous)

Favorites stored per user (uid)

Notes for each favorite city

CRUD operations (add / edit / delete)

Real-time updates (no manual refresh)

Optional: show current weather for favorites
Tech Stack

Swift, SwiftUI

Combine

URLSession

Open-Meteo API (no API key required)

Firebase iOS SDK

Authentication (Anonymous)

Realtime Database
